## 🚀 HOW TO RUN

### Prerequisites
- Node.js (for frontend)
- Python 3.12+ (for backend)
- Git

### Quick Start
```bash
# Terminal 1: Start Backend
cd flask_backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python simple_app.py

# Terminal 2: Start Frontend  
cd ..
npx vite dev
```
